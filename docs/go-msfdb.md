## Usage: go-msfdb on different server

```
$ go-msfdb server -bind=192.168.10.1 -port=1327
```

Sepcify URL of the go-msfdb in config.toml

```
[metasploit]
url = "http://192.168.0.1:1327"
```

```
$ vuls report 
```
